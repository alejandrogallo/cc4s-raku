# `cc4s-man`

Use `cc4s-man` from vim by adding this into your `~/.vimrc`

```vimrc
command! Cc4s :exec "!cc4s-man ".expand("<cword>")
```

and put your cursor on an algorithm name and hit the command
`:Cc4s`.
